# [Ludum dare 46](https://ldjam.com/)
Theme: Keep it alive  

![Cover](Screenshots/ZooKeeperLong.png)  
This game is about abondoned Zoo, and you are a new zookepeer.  
You need to entertain, feed and take care of various animals.
